About

While doing pure mathematics, I personally studied the theory of machine learning, did some experiments by myself and wrote some related blog posts. For example, 1, 2. Personal projects

1. gann

gann is a library for approximate nearest neighbor search purely written in golang. gann stands for Go-Approximate-Nearest-Neighbor. At the time when I launched this project, there is no libraries for ANN search written in go except for the ones which internally call c++ implemented APIs. That's why I started to develop gann so that it is purely written in golang and independent of other programming languages. I wrote the blog post about the algorithm of gann: Approximate Nearest Neighbor search in Go

2. doogle

doogle is a Proof of Concept software of decentralized search engine based on gRPC written in Go.

3. intergo

intergo is a package for interleaving / multileaving ranking generation in go.

4. go-walker

go-waker is a tiny package for Walker's sampling method in Go.
