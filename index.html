Open source projects

1. gann

gann is a library for approximate nearest neighbor search purely written in golang. gann stands for Go-Approximate-Nearest-Neighbor. At the time when I launched this project, there is no libraries for ANN search written in go except for the ones which internally call c++ implemented APIs. So I wrote gann so that it is purely written in golang and independent of other languages.
